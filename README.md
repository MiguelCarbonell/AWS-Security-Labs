# AWS Security Labs

This repository documents my hands-on learning journey in cloud security using AWS.

Its purpose is to build practical skills in identity and access management, access control, secure storage, and AWS security best practices through hands-on labs.

---

## Labs Included

- [IAM Lab](./iam-lab)
- [S3 Lab](./s3-lab)

---

## Objective

To develop real-world cloud security skills by performing practical labs instead of relying only on theory.

---

## Skills Practiced

- AWS Identity and Access Management (IAM)
- Multi-Factor Authentication (MFA)
- Secure account setup
- Amazon S3 basics
- Secure cloud storage configuration
- Versioning and basic data protection
- Basic cloud security principles

---

## Progress So Far

So far, this repository includes practical work on:

- Secure AWS account setup
- IAM user creation and MFA configuration
- Creation of a private S3 bucket
- Uploading objects to S3
- Enabling bucket versioning
- Understanding the difference between EC2 instances and S3 storage
